Regular Show Wiki API

Bienvenidos al proyecto Regular Show Wiki API. Aquí podrás encontrar información relacionada a la serie animada creada por J.G. Quintel: personajes, episodios y cómics, además de un pequeño agregado sobre la película.

Este proyecto es sin fines de lucro y está dirigido a programadores novicios que buscan una API para practicar habilidades de programación. Inspirado por la API de Rick and Morty, esta API busca aportar un pequeño grano de arena al mundo del desarrollo.

La API está desarrollada en Python con FastAPI, ejecutable localmente con Uvicorn. En la Wiki/Docs encontrarás detalles de endpoints y ejemplos para jugar con ella. Abajo te dejo accesos directos:

Repositorio OpenAPI Docs de la Wiki
Nota: Esta web es una demostración de lo que se puede construir con la API y está en crecimiento (en especial la información de personajes).
